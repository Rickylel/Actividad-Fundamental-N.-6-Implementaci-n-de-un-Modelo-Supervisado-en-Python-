# Actividad-Fundamental-N.-6-Implementaci-n-de-un-Modelo-Supervisado-en-Python-

# Clasificación de Cáncer de Mama con Regresión Logística

Este repositorio contiene la implementación de un modelo de aprendizaje supervisado (Regresión Logística) para predecir si un tumor es benigno o maligno basándose en el **Breast Cancer Wisconsin (Diagnostic) Dataset**.

## Descripción de la Actividad
Proyecto para la asignatura de **Programacion para Inteligencia Artificial**.
El objetivo es demostrar el uso de técnicas de preprocesamiento (Estandarización) y la evaluación de modelos de clasificación en Python.

## Tecnologías utilizadas
* Python 3
* Scikit-learn (Modelado y Dataset)
* Pandas (Manipulación de datos)
* Matplotlib & Seaborn (Visualización)

## Cómo ejecutar el código
